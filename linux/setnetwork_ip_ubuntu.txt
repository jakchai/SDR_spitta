Oracle VirtualBox
===========
1.NAT เป็น Network ประเภท Network Address Translation โดยตัว Guest OS จะกำหนดให้มีตัว IP Address ขึ้น1 ชุด ซึ่งตัว IP Address 
ที่ได้มานั้นจะไม่ได้ไปเอามาจากตัว DHCP Server แต่ตัว VirtualBox ได้มีการกำหนดให้กับตัว Guest OS เองตามตัว Network Interface Card 
ที่ใช้งาน ซึ่ง Guest OS สามารถออกสู่ INternet ได้เหมือนกับเครื่อง Host OS โดยจะมีการควบคุมจากตัว VirtualBox's NAT engine โดยตัว NAT 
จะนิยมใช้กันมากโดยสามารถเข้าไปตั้งค่าได้ ดังรูป

2.Bridged Adapter เป็น Network ที่เชื่อมต่อกันระหว่าง Guest OS กับ Network จริงที่ Host OS ใช้งานอยู่ โดยตัว Guest OS จะมองว่าเป็นส่วนหนึ่งของ
 Network จริง หรือเป็นเครื่อง Server ที่มีอยู่จริงใน Network หากมีการแจกจ่าย IP Address ผ่าน DHCP Server ตัว Guest OS ก็จะได้รับตัว
 IP Address ไปด้วย ซึ่งเหมาะที่จะนำไปใช้ในการทดสอบในระบบสภาพแวดล้อมจริง โดยสามารถเข้าไปเลือกได้ดังรูป
