Using Wine
==========
sudo apt install wine

1. Install Wine:
Open a terminal and run sudo apt install wine to install the Wine compatibility layer. 
2. Download the Windows App:
Download the .exe installer file for the Windows application you want to run. 
3. Run the Installer:
From the file manager: Right-click the .exe file and select "Open With > Wine Windows Program Loader". 
From the terminal: Navigate to the download directory in the terminal using cd and then run wine your_application_installer.exe.

install boot and active driver
------------------------------
 
source /opt/PetaLinux/settings.sh
cd /home/user01/SW/
git clone https://github.com/analogdevicesinc/meta-adi.git
export dts_to_use=zynq-zed-adv7511-ad9361-fmcomms2-3
petalinux-create project --template zynq -n zed_fmcomm2 --force
mkdir cache
mkdir cache/downloads
mkdir cache/sstate-cache
cd zed_fmcomm2/
echo "KERNEL_DTB=\"${dts_to_use}\"" >> project-spec/meta-user/conf/petalinuxbsp.conf
nano project-spec/meta-user/conf/petalinuxbsp.conf
**ในคำสั่งนี้ copy pathสองบรรทัดดานลางไปวางไว **
SSTATE_DIR = "/home/user01/SW/cache/sstate-cache"
DL_DIR = "/home/user01/SW/cache/downloads"
**จากนั้น Write Out และ Exit**

petalinux-config --get-hw-description=/home/user01/meta-adi/system.xsa

***System will allow us to set up, if we miss it so we use command petalinux-config
for later set up.
>>Yocto Setting
Add /home/user01/SW/meta-adi/meta-adi-xilinx to "user layer"
Add file:////home/user01/SW/mirror/rel-v2024.1 to pre-mirror
Update parallel thread to 16 สำหรับเครื่องที่ออฟฟต
petalinux-config -c kernel
petalinux-config -c rootfs
petalinux-build

cd images/linux/
petalinux-package --boot --fsbl zynq_fsbl.elf --fpga system.bit --u-boot --force
